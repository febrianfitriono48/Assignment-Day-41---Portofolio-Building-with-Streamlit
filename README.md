# Nickel Production Intelligence Dashboard

Interactive Streamlit dashboard for analyzing global nickel production trends.

## Features
- Country filtering
- Time range selection
- Continent aggregation
- 5-year production trend
- KPI metrics
- Business insights visualization

## Tech Stack
- Python
- Pandas
- Plotly
- Streamlit
